NU Olympic GAME
-----------------------------------------------------------------------------------
เกมวิ่งหลบหลีก ใช้ Computer Vision จาก MediaPipe ในการตรวจจับท่าทางของผู้เล่น เพื่อควบคุมตัวละครผ่านกล้องเว็บแคม!

🎮 วิธีการเล่น (How to Play)
-----------------------------------------------------------------------------------
การควบคุม:
ใช้ร่างกายของคุณขยับไป ซ้าย หรือ ขวา เพื่อเลื่อนตำแหน่งตัวละครให้ตรงกับเลนที่ต้องการ
ระบบจะทำการตรวจจับตำแหน่งของจมูกและมือ เพื่อคำนวณการเคลื่อนที่ของตัวละครโดยอัตโนมัติ

เป้าหมาย:
พยายามใช้มือเก็บกิ้งก่าง(Obstacles)เพื่อทำคะแนน และหลบรถ(Enemies) ให้ได้นานที่สุด
ยิ่งทำคะแนนได้มาก ความเร็วของเกมจะค่อยๆ เพิ่มขึ้น

กติกา:
หากตัวละครชนเข้ากับรถ พลังชีวิต(Lives) จะลดลง
เกมจะจบลง (Game Over) เมื่อพลังชีวิตหมด
เมื่อจบเกม สามารถดูคะแนนของคุณและเทียบอันดับในตารางผู้นำ(Leaderboard)

⌨️ ปุ่มควบคุม (Controls)
-----------------------------------------------------------------------------------
เริ่มเกม: ตามที่ระบุในหน้า Start Screen

Game Over:    

กด 'H': กลับไปหน้าหลัก (Home)

กด 'X': ออกจากเกม (Exit)

🛠 ความต้องการของระบบ (Requirements)
-----------------------------------------------------------------------------------
Python 3.x

Libraries: pygame, opencv-python, mediapipe, numpy

Hardware: กล้องเว็บแคมที่ใช้งานได้ปกติ

🚀 วิธีเริ่มต้นใช้งาน
-----------------------------------------------------------------------------------
ตรวจสอบให้แน่ใจว่าติดตั้ง library ครบถ้วน:

Bash   pip install pygame opencv-python mediapipe numpy


รันโปรแกรมผ่านไฟล์หลัก:

Bash   python main.py
